Meet the Builders

Five outputs from one unified definition.

🤖

OpenAI

Native OpenAI format. Optimized for GPT-4, Claude, Gemini compatibility.

{ "system": "...", "messages": [...] }
🧠

Anthropic

Anthropic Messages API format. System prompt + messages.

{ "model": "claude-3-opus", "system": "...", "messages": [...] }
⚡

AWS Bedrock

Bedrock ConversationXML. Multi-model inference ready.

<messages> <system>...</system> <message>...</message> </messages>
🔧

Replicate

Replicate format. Input/output optimized for structured tasks.

{ "prompt": "...", "input_params": {...} }
🎯

Custom

Roll your own. Extend with custom builders for proprietary tools.

class MyBuilder(Builder): def build(self, ir): return my_format(ir)

Up and Running in 2 Minutes

Copy. Paste. Ship.

1. Install

pip install promptosaurus

2. Define Your Prompt

system: "You are an AI assistant"
user: "What is the meaning of life?"
examples:
  - role: assistant
    content: "The meaning of life is..."

3. Build All 5 Outputs

from promptosaurus import Promptosaurus

p = Promptosaurus('prompt.yml')

openai_prompt = p.to_openai()
anthropic_prompt = p.to_anthropic()
bedrock_prompt = p.to_bedrock()
replicate_prompt = p.to_replicate()
custom_prompt = p.to_custom()
